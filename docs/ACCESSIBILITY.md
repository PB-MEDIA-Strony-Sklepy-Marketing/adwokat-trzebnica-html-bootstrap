# Accessibility Guide

## WCAG 2.1 Level AA Compliance

### Requirements
- Semantic HTML
- Alt text for images
- ARIA labels
- Keyboard navigation
- Focus indicators
- Color contrast
- Screen reader support

### Testing
```bash
npm run test:a11y
```

### Manual Testing
- Tab through page with keyboard
- Test with screen reader (NVDA/JAWS)
- Check color contrast ratios
