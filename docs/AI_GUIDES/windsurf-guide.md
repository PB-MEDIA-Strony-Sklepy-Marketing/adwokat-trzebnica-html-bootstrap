# Windsurf AI Guide - Kancelaria Adwokacka Website

## About Windsurf
Windsurf is an AI coding assistant with powerful multi-file editing capabilities.

## Key Features
- **Flow Mode**: Multi-step, multi-file changes
- **Cascade**: Complex refactoring across codebase
- **Real-time Preview**: See changes immediately

## Project Setup
1. Open project in Windsurf
2. Load .windsurfrules configuration
3. Start dev server: `npm run dev`
4. Use preview pane for real-time feedback

## Strengths
- Multi-file refactoring
- Component creation
- CSS styling with live preview
- Form implementation
