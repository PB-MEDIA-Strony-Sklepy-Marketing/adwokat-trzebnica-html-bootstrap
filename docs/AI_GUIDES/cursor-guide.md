# Cursor AI Guide - Kancelaria Adwokacka Website

## Cursor Overview
Cursor is an AI-powered code editor built on VS Code with native AI integration for code generation and editing.

## Quick Start
1. Open project in Cursor
2. Read `.cursorrules` file (auto-loaded)
3. Use Cmd/Ctrl+K for inline edits
4. Use Cmd/Ctrl+L for chat

## Key Features for This Project
- **Inline editing**: Cmd/Ctrl+K to edit selected code
- **Chat**: Cmd/Ctrl+L for questions and complex tasks
- **Multi-file edits**: Edit across multiple files simultaneously
- **Code completion**: Tab to accept suggestions
- **Terminal integration**: Run npm commands directly
